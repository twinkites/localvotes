Data Sources

  • ProPublica Congress API — federal senators & House reps, voting records, committees
  • OpenStates API — state legislators by location
  • Wikidata SPARQL — statewide executives (Governor, AG, and more)
  • US Census Geocoder — congressional district lookup & reverse geocoding
  • Census TIGERweb — district boundaries, school district & county lookup
  • Nominatim (OpenStreetMap) — reverse geocoding fallback
  • FEC Open Data — federal campaign finance
  • Congress.gov API — sponsored legislation
  • Zippopotam.us — ZIP → coordinates
